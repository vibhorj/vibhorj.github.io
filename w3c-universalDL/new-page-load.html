type dataLayer in browser console

1. actual pageload :: each time a new page loads (excl. new page renders in SPA context)


	dataLayer = window.dataLayer || [];
	dataLayer.push({
		"pageInstanceID": "dev", // "dev", "staging", "prod"
		"page": pageObj,
		"user": userObj
	});

2. virtual page :: new page renders in SPA context ---> just update the new page object with new context, together with additional key _clear: true 


	dataLayer = window.dataLayer || [];
	dataLayer.push({
		"_clear": true,
		"page.pageInfo": pageObj.pageInfo coresponding top new context
		"page.category": pageObj.category coresponding top new context
		"page.attributes": pageObj.attributes coresponding top new context
	});
